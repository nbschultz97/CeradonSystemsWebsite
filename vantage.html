Frequently Asked Questions

How does passive WiFi sensing work? +

WiFi signals from existing access points pass through walls and reflect off objects — including human bodies. When someone moves, breathes, or changes position, the WiFi signal characteristics (Channel State Information, or CSI) change measurably. Vantage captures these CSI changes passively — it only listens to WiFi frames, never transmits. Machine learning models then classify the patterns to detect presence and movement.

What types of walls can it see through? +

WiFi CSI works through most common building materials: drywall, wood framing, concrete block, brick, and glass. Performance degrades through metal-lined walls, wire mesh, or heavily reinforced concrete. The 2.4 GHz band is used by default for better wall penetration compared to 5 GHz. Real-world performance varies by wall composition, thickness, and ambient WiFi density.

What is the detection range? +

Detection range depends on the ambient WiFi environment and wall composition. In typical indoor environments with standard construction, presence detection has been demonstrated at room-scale distances (3–8 meters through walls). Range extends with stronger ambient WiFi signals and directional antennas. We're conducting field validation to publish verified range specifications.

Does it integrate with ATAK? +

ATAK integration is on the development roadmap. The planned implementation uses Cursor-on-Target (CoT) XML messages over multicast, allowing Vantage presence data to appear as map overlays in ATAK/WinTAK. The JSON output format is already structured to support this conversion. Contact us if ATAK integration is a priority for your evaluation.

How is this different from Lumineye or Camero? +

Lumineye and Camero use active radar — they transmit RF energy that bounces off targets. This works well but has two major drawbacks: (1) the emission is detectable by electronic warfare systems, compromising your position, and (2) the hardware is expensive ($5K–$85K). Vantage uses passive WiFi CSI — it only listens to existing WiFi signals and never emits. This makes it completely covert and dramatically cheaper since it runs on commodity WiFi hardware.

Does it need an existing WiFi network? +

Vantage works best with ambient WiFi signals from existing access points. In environments without WiFi infrastructure, a covert transmitter node can be deployed — but this trades some of the passive advantage. Research into exploiting other ambient RF sources (cellular, IoT) is ongoing.
