Vantage Platform

Zero-Emission Detection. Human truth through any wall.

Vantage uses passive Wi-Fi CSI to detect humans and estimate pose through walls without emitting RF energy. Zero-emission detection stays resilient in contested EW environments while keeping teams low-signature for breach operations, site security, and covert overwatch.

At a fraction of the cost of active radar systems, Vantage delivers covert through-wall awareness on COTS edge hardware — deployable in minutes, not hours.

Capabilities

Each module is engineered around the pose estimation pipeline—supporting features feed richer skeletal overlays and mission telemetry while staying passive.

Presence Detection

Low-latency detection on sub-second windows with adaptive thresholds for high-clutter structures.

Pose States

Standing, crouched, and prone classification primes the skeletal estimator and informs ROE.

Motion Vectors

Vectorized CSI deltas map speed, direction, and acceleration to anticipate movement.

Zone / Sector Location

Triangulate presence into sectors for rapid room clearing and follow-on skeletal overlays.

Behavioral Cues

Micro-Doppler patterns reveal agitation, compliance, and dwell time.

Local Logs

Architecture

From RF capture to pose inference, every stage runs locally. CSI ingest → denoising → skeletal estimation → redacted telemetry.

  • ▹ Modular pipeline orchestrated with containerized microservices.
  • ▹ GPU-optional inference with quantized pose networks for Jetson / Pi.
  • ▹ Secure local API for overlays, dashboards, and third-party integrations.

Operational Stack

Built for embedded teams that need deterministic behavior. Every module stays lean so future skeletal overlays render without starving CSI ingest.

RF Front-End

Intel 5300 / AX210 profiles, calibrated antenna maps, and OUI-cached spectrum intel for quick device attribution.

Pose Core

WiPose / PassiveWiFiPose-ready tensors with quantized weights to keep Jetson Nano and Pi 5 within thermal limits.

Telemetry

Schema-validated JSON, CSV, and SQLite logs with hash-chained records for chain-of-custody.

Visualization

Reserved HUD bands for skeletal overlays, mobility vectors, and direction/distance labels without redrawing the full scene.

Hardening & Ops

  • ▹ Offline update bundles signed for air-gapped kits.
  • ▹ Configurable emissions lock to keep radios passive during staging.
  • ▹ CLI health checks for CSI packet loss, clock drift, and thermal margins.
  • ▹ Export profiles for ATAK, KML, and offline replay without cloud calls.
