Vantage Platform

How passive WiFi through-wall detection works

Vantage exploits WiFi Channel State Information (CSI) ‚Äî the subtle signal distortions caused by human bodies in a WiFi field. When someone moves, breathes, or shifts position behind a wall, the WiFi signal changes. Vantage detects those changes passively.

1. Listen

Captures existing WiFi frames from ambient access points. No transmitter needed.

2. Extract

Pulls CSI amplitude and phase data across subcarriers using commodity WiFi chipsets.

3. Classify

ML models process conditioned signals to detect presence, movement, and activity patterns.

4. Report

Outputs presence alerts via JSON, CLI, or ATAK CoT feed in real time.
